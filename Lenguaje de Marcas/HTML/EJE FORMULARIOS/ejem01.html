Elementos o Etiquetas de los formularios

Elementos input y label

En input de tipo texto definimos name que es el nombre de la variable que usamos cuando enviamos el formulario, y el id que es el atributo que identifica que este input y luego poddemos usar la etiqueta label. En label usamos el atributo for para indicar a que input pertenece.

Elemento fieldset

Permite agrupar un conjunto de campos de un formulario de forma visula mediante un marco y de forma funcional permitiendo realizar acciones sobre todos los elementos agrupados. Podemos usar la etiqueta "legend" para poner un texto de título al marco.
