## 📖 Descripción del Proyecto
Este repositorio contiene todo el material de clase organizado para las asignaturas de **Lenguaje de Marcas** y **Entornos de Desarrollo**. El objetivo es aplicar el control de versiones con Git y el trabajo colaborativo en GitHub.

## 📂 Estructura del Repositorio
1. **Apuntes de teoría**: Resúmenes propios y mapas conceptuales.
2. **Ejercicios**: Actividades realizadas y soluciones documentadas.
3. **Prácticas**: Código fuente y explicaciones técnicas.
4. **Software necesario**: Listado de herramientas y guía básica de instalación.

## 🛠️ Software Necesario
- **Git**: sistema de control de versiones utilizado para gestionar los cambios del repositorio.  
  Enlace: https://git-scm.com/  
  Instalación básica: descargar el instalador desde la web oficial, ejecutarlo y comprobar la instalación con el comando `git --version`.

- **Visual Studio Code**: editor de código recomendado para editar archivos del proyecto.  
  Enlace: https://code.visualstudio.com/  
  Instalación básica: descargar el instalador, completar la instalación y abrir la carpeta del repositorio desde VS Code.

- **GitHub**: plataforma utilizada para alojar el repositorio y colaborar mediante ramas, commits y pull requests.  
  Enlace: https://github.com/  
  Uso básico: iniciar sesión, acceder al repositorio del grupo y revisar ramas, issues y pull requests.

- **Navegador web**: necesario para visualizar ejercicios y prácticas de Lenguaje de Marcas.  
  Opciones recomendadas: Google Chrome o Mozilla Firefox.  
  Instalación básica: descargar el navegador desde su web oficial e instalarlo siguiendo el asistente.

## 🚀 Instrucciones de Uso
1. **Clonar el repositorio**:  
   `git clone https://github.com/Daniela-20040103/apuntes-daw.git`
2. **Crear una rama propia para trabajar**:  
   `git checkout -b rama-tu-nombre`
3. **Subir cambios**:  
   `git add .`, `git commit -m "docs: mensaje descriptivo"` y `git push`
